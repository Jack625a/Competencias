SISTEMA DE CONTROL DE VERSIONES

  • Git Init: Inicializa nuestro repositorio
  • Git Add  Añade los cambios a la Zona Temporal o Zona de Espera
  • Git Commit -m "" Confirma los cambios de la zona tempral para la creacion de una nueva version de nuestro codigo
  • Git Status Nos muestra el estado en el que se encuentra nuestro repositorio
  • Git log Nos muestra el historial completo de nuestro repositorio
  • Git log --oneline Nos muestra un resumen de nuestro historial del repositorio
  • Git diff Va Vb Nos muestra la diferencia que existe entre una version de codigo con otra version de codigo
  • Git checkout Nos da la posibilidad de hacer una navegacion a una version anterior o tambien navegar entre ramificaciones
  • Git Pull. Nos obtine todos los cambios que se tiene en el repositorio remoto a nuestro repositorio local
  • Git Push Nos da la posibilidad de enviar nuestros cambios de nuestro repositorio local a nuestro repositorio remoto
  • Git Clone Nos da la posibilidad de clonar un repositorio remoto
  • Git Config Nos da la posibilidad de modifcar los atributos de nuestro repositorio
